## 说明

uview-max，是 uni-app 全面兼容 vue3/nvue 的 uni-app 生态框架，全面的组件和便捷的工具会让您信手拈来，如鱼得水。uview-max 是基于 uView2.x 移植的支持 vue3 的版本，感谢 uView。
